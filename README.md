# منظومة التوظيف الذكية متعددة الأبعاد

> تحويل التوظيف من عملية إدارية إلى رحلة حياة مهنية ذكية، مستدامة، ومتطورة — حيث يكون الفرد محور المنظومة، والذكاء الاصطناعي هو المرشد الحيوي.

نظام بيئي (Ecosystem) يوحّد جميع أطراف العملية التوظيفية (باحث، جهة، موصي، مدرب، مستشار) في منصة تفاعلية واحدة مدعومة بذكاء اصطناعي متعدد الأدوار.

## 🛠️ التقنيات المخطط لها (Planned Tech Stack)

| الطبقة | التقنية |
|--------|---------|
| إطار العمل | **Vue 3** (Composition API, `<script setup>`) |
| اللغة | **TypeScript** (الوضع الصارم) |
| أداة البناء | **Vite 5** |
| واجهة المستخدم | **Vuetify 3** |
| إدارة الحالة | **Pinia** |
| التوجيه | **Vue Router 4** |
| طلبات HTTP | **Axios** |
| النماذج والتحقق | **vee-validate 4** |
| التعدد اللغوي | **vue-i18n 9** (الافتراضي: العربية، RTL) |
| مدير الحزم | **Yarn 1.x** |

> راجع [`DOC/PROJECT_STRUCTURE.md`](DOC/PROJECT_STRUCTURE.md) للتفاصيل الكاملة عن البنية والاتفاقيات.

## 🚀 البدء (Getting Started)

> ملاحظة: لم تُضَف الشيفرة المصدرية للتطبيق بعد — يحتوي المستودع على الوثائق فقط في هذه المرحلة.

بمجرد إضافة هيكل التطبيق:

```bash
yarn install      # تثبيت الاعتماديات
yarn dev          # تشغيل خادم التطوير
yarn build        # بناء الإنتاج
yarn typecheck    # فحص الأنواع
yarn lint         # تدقيق الكود
```
